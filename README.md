# 🧠 Anatomy Learning Game

> 🎮 An interactive web-based learning game designed to help students understand human anatomy through quizzes, visual learning, and real-time feedback.

---

## ✨ Features

- 🧩 Interactive anatomy quiz system  
- 📊 Real-time score tracking  
- ✅ Instant right/wrong feedback  
- 📈 Result summary dashboard  
- 🖥️ Clean and responsive UI  
- ⚡ Fast performance (Vanilla JavaScript)

---

## 🛠️ Tech Stack

- 🌐 HTML5 – Structure  
- 🎨 CSS3 – Styling & UI Design  
- ⚙️ JavaScript (ES6) – Game logic & interactivity  

---

## 🚀 How to Run This Project

```bash
# 1. Clone the repository
git clone https://github.com/your-username/anatomy-learning-game.git

# 2. Open the project folder
cd anatomy-learning-game

# 3. Run the project
Open index.html in your browser
````
